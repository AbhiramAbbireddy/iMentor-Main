## L1

### Definition
L1 regularization, also known as L1 norm or Lasso regularization, is a technique used in machine learning to reduce overfitting and improve model interpretability by adding a penalty term to the loss function based on the absolute value of the coefficients. This penalty term encourages the model to use only the most important features, effectively eliminating irrelevant or redundant features. By promoting sparsity in the model parameters, L1 regularization can be used for feature selection.

### Intuition
The intuition behind L1 regularization is to penalize the model for using too many features, which can lead to overfitting. By adding a penalty term to the loss function, the model is encouraged to use only the most important features, which can be seen as a form of feature selection. This is particularly useful in applications where interpretability is important, such as in medical diagnosis or financial forecasting, where understanding the impact of each feature is crucial. For example, in a medical diagnosis model, L1 regularization can help identify the most important symptoms or risk factors that contribute to a particular disease, allowing for more accurate and interpretable predictions.

In real-world applications, L1 regularization can be thought of as a way to simplify complex models by eliminating unnecessary features. This can be particularly useful when dealing with high-dimensional data, where many features may be correlated or redundant. By applying L1 regularization, the model can automatically select the most important features, reducing the risk of overfitting and improving the overall performance of the model. Furthermore, the sparsity property of L1 regularization can also help to reduce the computational cost of the model, as fewer features need to be computed and stored.

### Mathematical Foundation
$$
J(\theta) = \text{loss}(\theta) + \lambda \sum_{i=1}^{n} |\theta_i|
$$

In this equation, $J(\theta)$ represents the total loss of the model, $\text{loss}(\theta)$ represents the original loss function, $\lambda$ is a hyperparameter that controls the strength of the penalty term, $\theta_i$ represents the $i^{th}$ model parameter, and $n$ is the total number of parameters. The penalty term, $\lambda \sum_{i=1}^{n} |\theta_i|$, is the absolute value of the magnitude of the coefficients, which encourages the model to use only the most important features.

### Worked Example

**Problem:** Suppose we have a linear regression model with three features, $x_1$, $x_2$, and $x_3$, and corresponding weights, $w_1$, $w_2$, and $w_3$. We want to apply L1 regularization with $\lambda = 0.1$ to the model.

**Solution:**
The original loss function is $\text{loss}(\theta) = \frac{1}{2} \sum_{i=1}^{n} (y_i - (w_1 x_{1i} + w_2 x_{2i} + w_3 x_{3i}))^2$. Applying L1 regularization, the new loss function becomes:
$$
J(\theta) = \frac{1}{2} \sum_{i=1}^{n} (y_i - (w_1 x_{1i} + w_2 x_{2i} + w_3 x_{3i}))^2 + 0.1 (|w_1| + |w_2| + |w_3|)
$$
If the penalty term causes one of the weights to become zero, that feature is effectively removed from the model.

### Key Takeaways
- L1 regularization adds a penalty term to the loss function based on the absolute value of the coefficients.
- The penalty term encourages the model to use only the most important features, promoting sparsity in the model parameters.
- L1 regularization can be used for feature selection, as it tends to eliminate irrelevant or redundant features.
- The strength of the penalty term is controlled by the hyperparameter $\lambda$.

### Common Misconceptions
- ⚠️ **Misconception:** L1 regularization always results in a sparse model. **Correction:** The sparsity of the model depends on the value of $\lambda$ and the data.
- ⚠️ **Misconception:** L1 regularization is the same as L2 regularization. **Correction:** L1 regularization uses the absolute value of the coefficients as the penalty term, while L2 regularization uses the square of the coefficients.
